# Ananke Gohugo Theme

Template PBN genere depuis https://github.com/theNewDynamic/gohugo-theme-ananke.git.

## Validation

- Slug PBN: `gohugo-theme-ananke`
- Theme Hugo: `gohugo-theme-ananke`
- Profil: `blog`
- Contrat contenu: `content/posts/{{slug}}.md` -> `/posts/{{slug}}/`
- Version Hugo: `0.160.0`
- Build: `hugo --gc --minify`
- Publication: `public`

Avant activation production, lancer un build Hugo avec un contenu de test.
